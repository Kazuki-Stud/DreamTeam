## Losties

A website for users to post and find lost items. People can search for their property by selecting a category and using keywords.

## Functionality
* **Post Creation**: Authorized users fill forms with summary, keywords, address, and type.
* **Legal Disclaimer**: Shows a modal with instructions before form entry.
* **Status Control**: Manual updates to Pending, Closed, or Deleted.
* **User Profile**: List of posts with status tracking and match history.
* **Post Editing**: Tools to modify or remove active listings.
* **Losing Feed**: Display of cards for guests and users.
* **Field Privacy**: Settings for visibility: All, Authorized, or Match only.
* **Data Masking**: Hiding details in search to prevent copying or scams.
* **Search Cleanup**: Removal of closed or deleted items from results.
* **Comparison Engine**: Automatic links between posts using keywords and location.
* **Probability Scale**: Calculation of match accuracy.
* **Match Notifications**: Email and interface alerts for authors.
* **Direct Links**: Navigation from alerts to item details.
* **Contact Exchange**: Transfer of emails after mutual confirmation.
## Technology

- Backend: Java, Spring Boot, Spring security
- Frontend: Vue.js + TypeScript
- DB: PostgreSQL
- CI/CD: GitHub Actions
