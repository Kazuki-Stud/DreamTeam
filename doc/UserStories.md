US-001: Found Item Posting
As an authorized user
I want to fill out a found item form
In order to inform other system participants about the find

[ ] "+" button on the home page to navigate to the form
[ ] Display of a modal window with legal instructions before filling
[ ] Input fields: Summary, Keywords, Address, Type of item
[ ] Restriction of post creation access for unauthorized persons

US-002: Advertisement Data Privacy Management
As an advertisement author
I want to configure the visibility of individual data fields
In order to protect information from scammers and unauthorized persons

[ ] Visibility settings for each field Visible to: All, Authorized, Match only
[ ] Hiding sensitive information in general search according to set parameters
[ ] Blocking full access to item details to prevent unauthorized data copying

US-003: Automatic Matching System
As a system
I want to match data from lost and found advertisements
In order to suggest the most probable item return options to users

[ ] Comparison algorithm based on keywords and address
[ ] Calculation of an ownership probability scale based on input data matching
[ ] Dynamic update of the matches list when new posts are added

US-004: Match Notifications
As a lost item advertisement author
I want to receive notifications about found matches
In order to promptly verify information about a possible find

[ ] Sending an Email notification with a link to the found object
[ ] Display of an internal notification within the site interface
[ ] Navigation from the notification to a separate page with found item details

US-005: Contact Information Exchange
As an authorized user
I want to confirm a match and receive another user's contact information
In order to complete the item return process outside of the platform

[ ] Confirmation button for both parties
[ ] Transfer of the Email address to the counterparty after mutual confirmation
[ ] Logging of the contact transfer fact in the system

US-006: Advertisement Life Cycle Management
As an advertisement author
I want to change the status of my post
In order to update information in the system

[ ] Support for three system states: Pending, Closed, Deleted
[ ] Manual closure function for advertisements after the item is returned
[ ] Exclusion of advertisements with Closed and Deleted statuses from search results and matching algorithms

US-007: User Profile
As an authorized user
I want to have access to a profile with a list of my posts
In order to track advertisement status and incoming notifications

[ ] List of all user-created advertisements with current status indication
[ ] Section for viewing match notification history
[ ] Ability to navigate to edit or delete posts from the profile

US-008: Viewing and Searching Advertisements
As a guest or authorized user
I want to view the lost items feed
In order to personally identify items I have found

[ ] Display of advertisement cards on the home page or in search
[ ] Restriction of detailed information display according to privacy settings
[ ] Redirect to the authorization form when attempting to interact with creation or contact functions
