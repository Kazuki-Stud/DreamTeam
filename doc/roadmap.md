## Roadmap

### MVP 
* **Auth System**: Sign-up and login for post creation.
* **CRUD**: Forms for Lost and Found items.
* **Legal Warning**: Modal with instructions before first post.
* **Privacy**: Settings for field visibility (All, Authorized, Match only).
* **Matching Logic**: Keywords and Address comparison.
* **Status Control**: Toggle between Pending, Closed, and Deleted.
* **Profile Page**: Basic list of user posts and statuses.
* **Contact Exchange**: Secure email transfer after mutual consent.
* **Notification System**: Email alerts and site notifications for matches.
---
### v1.1
* **Probability Scale**: Visual indicator of match accuracy.
* **Bureau Integration**: API connections to local lost and found offices.
* **Computer Vision**: Image matching for similarity and item state.
* **Text Extraction**: Auto-filling forms from photos.
* **Interactive Maps**: Visual search for items by geofencing.
* **Unique ID Tracking**: Specific workflows for tech (IMEI, Serial numbers).
